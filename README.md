# Instructions

## Frontend

```
cd app
```

### Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn dev
```

## Backend

Before continue install task [https://taskfile.dev/#/usage]

```
brew install go-task/tap/go-task
```

```
cd backend
```

### Init environment

```
task init_env
```

### Install dependencies

```
task install_dependencies
```

### Run server

```
task run
```

### Run tests

```
task tests
```
